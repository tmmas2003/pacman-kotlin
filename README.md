# Pac-Man - Kotlin

A desktop implementation of the classic **Pac-Man** game developed in **Kotlin** as part of my university coursework.

---

## Overview

This project recreates the classic Pac-Man arcade game using Kotlin.

The player navigates Pac-Man through a maze, collecting points while avoiding ghosts. The game demonstrates object-oriented design, game loops, collision handling, and maze navigation.

---

## Features

- Player-controlled Pac-Man
- Ghost movement
- Maze navigation
- Collision detection
- Score collection
- Game loop implementation
- Keyboard controls

---

## Technologies

- Kotlin
- Object-Oriented Programming
- Game Loop
- Collision Detection

---

## Project Structure

```
Main.kt      -> Application entry point
Game.kt      -> Main game logic
View.kt      -> Rendering
Maze.kt      -> Maze generation and layout
Grid.kt      -> Grid management
Pacman.kt    -> Player behaviour
Ghosts.kt    -> Ghost AI and movement
```
